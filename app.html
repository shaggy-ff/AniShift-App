AniShift App Update Guide 🚀

වෙබ් කෝඩ් එක වෙනස් කරාට පස්සේ, අලුත් APK එකක් හදාගන්න ලේසිම පියවර 3 මෙන්න.

පියවර 1: කෝඩ් එක වෙනස් කරලා Save කරන්න

  • ඔයාගේ Editor එකේ (Antigravity/VS Code) සාමාන්‍ය විදිහටම වෙබ් කෝඩ් එක (HTML/CSS/JS) වෙනස් කරන්න.
  • වෙනස් කරපු හැම ෆයිල් එකක්ම අනිවාර්යයෙන්ම Save (Ctrl+S) කරන්න අමතක කරන්න එපා.

පියවර 2: අලුත් කෝඩ් එක GitHub එකට යවන්න

ඔයාගේ Editor එකේ යට තියෙන Terminal එක ඕපන් කරලා, මේ කමාන්ඩ් 3 පිළිවෙලට ගහලා Enter ඔබන්න:

git add .

git commit -m "Update App Code"

git push origin main

* (සමහරවිට main වෙනුවට master කියලා ගහන්න වෙයි, ඔයා කලින් පාවිච්චි කරපු එක අනුව).

පියවර 3: අලුත් APK එක Download කරගන්න

  • බ්‍රවුසර් එකෙන් ඔයාගේ GitHub රෙපොසිටරි පිටුවට යන්න.
  • උඩ මෙනු එකෙන් "Actions" කියන ටැබ් එකට යන්න.
  • කහ පාට රවුම කැරකිලා බිල්ඩ් වෙනවා පෙනෙයි. විනාඩි 3කින් විතර ඒක කොළ පාට (✅) වුණාම ඒක උඩ ක්ලික් කරන්න.
  • යටම තියෙන "Artifacts" කොටසින් AniShift-App-APK කියන එක Download කරගන්න. ඒක ඇතුළේ ඔයාගේ අලුත් ඇප් එක තියෙනවා!
වැදගත්: මේ විදිහට කරද්දි ඔයාගේ PC එකට කිසිම බරක් නෑ. App එක Build වෙන හැමදේම වෙන්නේ GitHub Cloud සර්වර් එක ඇතුළේ!
